PySpark Interview Questions and Answers

1What is Apache Spark? ▼
Apache Spark is an open-source cluster computing framework designed for fast and general-purpose large-scale data processing with in-memory computation capabilities.
2What is PySpark? ▼
PySpark is the Python API for Apache Spark, allowing Python developers to interface with Spark’s core functionalities and API seamlessly.
3What are RDDs in Spark? ▼
RDD (Resilient Distributed Dataset) is a fault-tolerant, immutable distributed collection of objects that can be processed in parallel.
4What are the key features of Apache Spark? ▼
  • In-memory computation for faster processing.
  • Support for multiple languages: Scala, Java, Python, and R.
  • Rich set of libraries including SQL, streaming, machine learning, and graph processing.
  • Lazy evaluation of transformations.
  • Fault tolerance with lineage graphs.
5What is the difference between RDD, DataFrame, and Dataset? ▼
RDD is the basic low-level API representing distributed collections. DataFrame is a distributed collection organized into named columns, with optimizations and schema. Dataset is a typed extension of DataFrames, combining benefits of RDDs and DataFrames with compile-time type safety (in Scala and Java only).
6What is SparkContext? ▼
SparkContext is the entry point to Spark functionality, responsible for connecting to the cluster manager and coordinating job execution.
7What is lazy evaluation in Spark? ▼
Lazy evaluation means Spark postpones computation until an action is invoked, optimizing the overall data processing workflow.
8What are transformations and actions in Spark? ▼
Transformations are operations on RDDs that return new RDDs (lazy). Actions trigger execution and return results or write data.
9How does Spark handle fault tolerance? ▼
Spark uses lineage information to recompute lost data partitions from parent RDDs in case of node failures.
10What is the difference between cache() and persist()? ▼
cache() is a shorthand for persist() with the default storage level (memory only). persist() allows specifying storage levels like MEMORY_AND_DISK.
11What is a broadcast variable? ▼
A broadcast variable allows the program to efficiently send a large read-only value to all worker nodes.
12What is an accumulator? ▼
An accumulator is a variable that tasks can add to using commutative and associative operations, used for counters and sums.
13What is Spark SQL? ▼
Spark SQL is a Spark module for structured data processing using DataFrames and SQL queries.
14What is a DataFrame? ▼
A DataFrame is a distributed collection of data organized into named columns, similar to a table in a relational database.
15How do you create a DataFrame in PySpark? ▼
You can create a DataFrame by loading data from external sources (like JSON, CSV, Parquet) or by converting an existing RDD or Python list using sparkSession.createDataFrame().
16What are the common file formats supported by Spark? ▼
CSV, JSON, Parquet, ORC, Avro, and text files are commonly supported.
17What is the difference between narrow and wide transformations? ▼
Narrow transformations (like map, filter) do not require data shuffling, while wide transformations (like reduceByKey, join) require shuffling data across partitions.
18What is a shuffle operation in Spark? ▼
Shuffle is a costly operation where data is redistributed across partitions, usually happening in wide transformations.
19What is DAG in Spark? ▼
DAG (Directed Acyclic Graph) is Spark’s execution plan representing sequence and dependencies of transformations.
20Explain Spark Streaming. ▼
Spark Streaming processes live data streams by dividing data into micro-batches for near real-time analytics.
21How to handle skewness in Spark? ▼
Use salting keys, broadcast joins, or repartition to handle skewed data.
22What is Tungsten in Spark? ▼
Tungsten is a Spark optimization project focusing on memory management and code generation for better execution efficiency.
23What is Catalyst Optimizer? ▼
Catalyst is Spark SQL’s query optimizer that analyzes and optimizes logical plans for efficient execution.
24What is the difference between coalesce() and repartition()? ▼
coalesce() reduces the number of partitions without shuffle (efficient for decreasing partitions), while repartition() can increase or decrease partitions and does shuffle.
25What is a checkpoint in Spark? ▼
Checkpointing saves the RDD to reliable storage to truncate lineage and recover data in case of failures.
26Explain the role of Spark driver and executor. ▼
The driver program manages the SparkContext and plans jobs. Executors run tasks and perform computations on worker nodes.
27How do you optimize joins in Spark? ▼
Use broadcast joins for small tables, partition data properly, and avoid skew by salting keys.
28What is a SparkSession? ▼
SparkSession is the unified entry point to Spark functionalities introduced in Spark 2.x, superseding SparkContext for SQL, DataFrame, and Dataset APIs.
29How is data serialized in Spark? ▼
Spark serializes data using Java serialization by default; Kryo serializer is an alternative that is faster and more efficient.
30What is MLlib? ▼
MLlib is Spark’s scalable machine learning library supporting classification, regression, clustering, and collaborative filtering.
31How do you run PySpark locally? ▼
Set master URL to 'local[*]' in SparkConf or submit spark-shell with --master local option.
32What is the difference between map() and flatMap()? ▼
map() applies a function to each element returning one output per input. flatMap() can return zero or more outputs per input, flattening the result.
33How do you persist RDDs? ▼
By calling persist() with a desired StorageLevel like MEMORY_ONLY or MEMORY_AND_DISK.
34What is the difference between a DataFrame and a table in Spark? ▼
A DataFrame is an in-memory distributed collection, whereas a Spark SQL table is a metadata abstraction representing data stored externally or registered temporarily.
35How do you read a CSV file in PySpark? ▼
Use spark.read.csv('filename.csv', header=True, inferSchema=True) to load a CSV file as a DataFrame.
36How can you view the schema of a DataFrame? ▼
Call df.printSchema() to display the hierarchical schema of the DataFrame.
37How do you select columns from a DataFrame? ▼
Use df.select('col1', 'col2') to select columns in PySpark.
38How do you filter rows in a DataFrame? ▼
Utilize df.filter(df['col'] > 5) or df.where(df['col'] == 'value') to filter rows.
39How do you perform groupBy operations? ▼
Use df.groupBy('col').agg({'col2':'sum'}) or similar aggregation functions.
40What are window functions in Spark? ▼
Window functions perform calculations across a set of rows related to the current row, useful for running totals, rankings, etc.
41What is explode in PySpark? ▼
The explode() function flattens arrays or maps into separate rows.
42How to join two DataFrames? ▼
Use df1.join(df2, df1["id"] == df2["id"], "inner") for joins like inner, left, right, or outer.
43How do you handle missing data in PySpark? ▼
Use df.na.drop() to drop nulls, df.na.fill(value) to fill nulls, or df.na.replace() to replace specific values.
44How to write a DataFrame to a Parquet file? ▼
Use df.write.parquet('output_path') to store the DataFrame in Parquet format.
45What is Py4J? ▼
Py4J enables Python programs running in a Python interpreter to dynamically access Java objects in a JVM, used internally by PySpark to call Spark in JVM.
46What is partitioning in Spark? ▼
Partitioning controls how data is distributed across the cluster, affecting parallelism and performance. Use repartition() and coalesce() for adjusting partitions.
47How can you broadcast a variable in PySpark? ▼
Use sc.broadcast(value) to create a broadcast variable accessible to all nodes.
48What’s the use of withColumn in PySpark? ▼
withColumn() is used to add a new column or replace an existing column in a DataFrame.
49How do you remove duplicates from a DataFrame? ▼
Use df.dropDuplicates() or df.dropDuplicates(['col1', 'col2']) for a subset of columns.
50How to handle skewed data in PySpark? ▼
Use techniques like salting keys, broadcasting small tables, leveraging custom partitioning, or applying map-side reductions to manage data skew.

SQL Interview Questions and Answers

1What is SQL? ▼
SQL stands for Structured Query Language, designed for managing and manipulating relational databases.
2What are the different types of SQL statements? ▼
DDL (Data Definition Language), DML (Data Manipulation Language), DCL (Data Control Language), TCL (Transaction Control Language).
3What is a primary key? ▼
A primary key uniquely identifies each row in a table and cannot contain NULL values.
4What is a foreign key? ▼
A foreign key is a field in one table that refers to the primary key in another table, establishing a relationship between tables.
5What are joins in SQL? ▼
Joins combine rows from two or more tables based on a related column. Types include INNER, LEFT, RIGHT, and FULL OUTER JOIN.
6What is normalization? ▼
Normalization is the process of organizing data in a database to minimize redundancy and improve data integrity.
7What is denormalization? ▼
Denormalization is the process of adding redundant data to speed up read operations, sometimes at the cost of write performance and storage.
8What is an index in SQL? ▼
An index is a database object that improves the speed of data retrieval operations on a table at the cost of additional writes and storage.
9What are the differences between clustered and non-clustered indexes? ▼
A clustered index sorts and stores the data rows physically in the table, while a non-clustered index creates a separate structure for the index with pointers to the data rows.
10What is a view in SQL? ▼
A view is a virtual table based on the result of a SELECT query, stored in the database without holding actual data.
11What is a stored procedure? ▼
A stored procedure is a precompiled collection of one or more SQL statements that can be executed as a program.
12What is a trigger? ▼
A trigger is a procedural code that is automatically executed in response to certain events on a particular table or view, such as INSERT, UPDATE, or DELETE.
13What is a subquery? ▼
A subquery is a query nested within another SQL query, often used in WHERE or HAVING clauses.
14What is a correlated subquery? ▼
A correlated subquery depends on values from the outer query and is evaluated for each row processed by the outer query.
15What is the difference between WHERE and HAVING? ▼
WHERE filters rows before grouping, HAVING filters groups after aggregation.
16What does GROUP BY do? ▼
GROUP BY groups rows that have the same values in specified columns for aggregate functions like COUNT, SUM, AVG, etc.
17What is an aggregate function? ▼
Aggregate functions perform calculations on multiple values to return a single value, e.g., COUNT(), SUM(), AVG(), MAX(), MIN().
18How do you retrieve unique records in SQL? ▼
Use the DISTINCT keyword in your SELECT statement.
19What is a transaction? ▼
A transaction is a unit of work performed within a database management system against a database, treated as a single logical operation.
20What are the ACID properties? ▼
Atomicity, Consistency, Isolation, and Durability—properties that guarantee reliable processing of database transactions.
21What is a constraint? ▼
A constraint enforces rules at the table level, e.g., NOT NULL, UNIQUE, PRIMARY KEY, FOREIGN KEY, CHECK, DEFAULT.
22How do you prevent SQL injection? ▼
Use prepared statements, parameterized queries, and input validation to prevent SQL injection.
23What is UNION in SQL? ▼
UNION is used to combine the result sets of two or more SELECT statements, removing duplicates.
24What is UNION ALL? ▼
UNION ALL combines the result sets of two or more SELECT statements, including duplicates.
25What is the difference between DELETE and TRUNCATE? ▼
DELETE removes rows one at a time and can use a WHERE clause; TRUNCATE removes all rows and cannot be rolled back in some databases.
26What is a self-join? ▼
A self-join is a query in which a table is joined with itself, useful for hierarchical or recursive data relationships.
27What is a cross join? ▼
A cross join returns the Cartesian product of two tables, combining all rows from the first table with all rows from the second.
28How do you add a new column to an existing table? ▼
Use ALTER TABLE table_name ADD column_name datatype.
29How do you update data in a table? ▼
Use the UPDATE command with a SET clause: UPDATE table SET column = value WHERE condition.
30How do you delete a table? ▼
Use DROP TABLE table_name to remove the entire table structure and its data.
31What is a composite key? ▼
A composite key is a primary key composed of two or more columns to uniquely identify a row.
32What is an alias in SQL? ▼
An alias is a temporary name given to a table or column for the purpose of a particular SQL query, using the AS keyword.
33How can you limit the number of rows returned? ▼
Use the LIMIT or TOP keyword depending on the database (e.g., LIMIT 10 in MySQL/Postgres, TOP 10 in SQL Server).
34What is NULL in SQL? ▼
NULL represents missing or unknown data in a table cell.
35How do you test for NULL values? ▼
Use IS NULL or IS NOT NULL operators, e.g., WHERE column IS NULL.
36What is the difference between CHAR and VARCHAR? ▼
CHAR is a fixed-length string, while VARCHAR is variable-length. VARCHAR is more space efficient for varying-length data.
37What is a CTE (Common Table Expression)? ▼
A CTE is a temporary result set defined within the execution scope of a single SQL statement using the WITH clause.
38What are window functions? ▼
Window functions perform calculations across a set of rows related to the current row within the query result set.
39How do you get the second highest salary from an Employee table? ▼
Use a subquery: SELECT MAX(salary) FROM employee WHERE salary < (SELECT MAX(salary) FROM employee).
40What does the CASE statement do? ▼
The CASE statement provides conditional logic similar to if-else in SQL, returning values based on conditions.
41What is a materialized view? ▼
A materialized view is a database object containing the results of a query that are stored physically and can be refreshed periodically.
42What is the difference between SFW and SFWR (SELECT-FROM-WHERE[-RETURN]) queries? ▼
SFW is used in most relational SQL queries; SFWR is a logical extension in some systems for returning complex data.
43How can you copy data from one table to another? ▼
INSERT INTO table2 (columns) SELECT ... FROM table1 WHERE ....
44What is the difference between union and join? ▼
UNION combines result sets vertically (same columns), while JOIN combines tables horizontally based on relationships between them.
45How do you rename a table? ▼
Use ALTER TABLE old_name RENAME TO new_name.
46What are scalar vs. aggregate functions? ▼
Scalar functions return a single value per row, aggregate functions compute a single value from multiple rows.
47How do you find duplicate rows? ▼
Use GROUP BY with HAVING COUNT(*) > 1 on the relevant columns.
48What are the advantages of using indexes? ▼
Indexes speed up data retrieval but can slow down write operations and consume disk space.
49What is database schema? ▼
A schema is the organizational blueprint of the database structure, including tables, columns, relationships, indexes, and views.
50What does the EXPLAIN keyword do? ▼
EXPLAIN shows the execution plan of a SQL statement, helping optimize queries and understand their behavior.

CSS Interview Questions and Answers

1What is CSS? ▼
CSS (Cascading Style Sheets) is a language used for describing the look and formatting of documents written in HTML.
2What are the different types of CSS? ▼
Inline, internal (embedded), and external CSS.
3What is the difference between class selectors and ID selectors? ▼
Class selectors are reusable and start with a dot (.), while ID selectors are unique for a page and start with a hash (#).
4What is the CSS box model? ▼
The box model includes content, padding, border, and margin around HTML elements.
5How do you center a div horizontally? ▼
Set margin: 0 auto; and a width for the div.
6What is the default position value in CSS? ▼
static is the default position value for all elements.
7What is z-index and when is it effective? ▼
z-index controls stacking order of positioned elements (position other than static).
8What is the difference between em and rem units? ▼
em is relative to the parent font-size, rem is relative to the root element (html) font-size.
9How do you add a background image in CSS? ▼
Use background-image: url('image.jpg');
10What is specificity in CSS? ▼
Specificity determines which CSS rule is applied when multiple rules could apply to the same element, based on selectors' types.
11How do you make a list horizontal? ▼
Set display: inline; or use float: left; or display: flex; on the list items.
12What is Flexbox? ▼
Flexbox is a layout model in CSS3 that allows responsive alignment and distribution of space among items in a container.
13Explain the difference between visibility: hidden and display: none. ▼
visibility: hidden makes the element invisible but preserves its space; display: none removes it from layout and flow.
14How do you set a custom font in CSS? ▼
Use @font-face rule or font-family referencing web fonts or local fonts.
15What does the float property do? ▼
float lets elements move to the left or right, allowing text and inline elements to wrap around them.
16What is a pseudo-class? ▼
A pseudo-class defines a special state of an element, such as :hover, :active, :nth-child().
17What is a pseudo-element? ▼
A pseudo-element styles specified parts of an element, like ::before or ::after.
18How can you make rounded corners? ▼
Use the border-radius property, e.g., border-radius: 10px;
19How do you create a transition effect? ▼
Use the transition property, e.g., transition: background 0.5s ease;
20How do you use CSS Grid? ▼
Define a grid container with display: grid; and specify columns/rows using grid-template-columns and grid-template-rows.
21How do you select all direct children of an element? ▼
Use the child combinator >, e.g., div > p
22What is important in CSS? ▼
!important increases the specificity and overrides all previous declarations.
23How do you prevent a CSS file from being cached? ▼
Append a query string version to the file, e.g., style.css?v=2.
24How do you style the first letter or first line in text? ▼
Use the ::first-letter and ::first-line pseudo-elements.
25What is CSS Sprites? ▼
CSS Sprites combine multiple images into a single image to reduce HTTP requests; background-position is used to display portions.
26What is a media query? ▼
A media query applies styles based on device characteristics like width, height, or orientation, enabling responsive web design.
27What is the difference between inline and block elements? ▼
Inline elements don’t start on a new line and only occupy the necessary width, while block elements begin on a new line and take full width available.
28What does overflow: hidden do? ▼
It cuts off content that exceeds an element’s box instead of letting it spill out.
29How do you vertically align an element in CSS? ▼
Use vertical-align for inline/inline-block or align-items: center with Flexbox, or display: table-cell and vertical-align:middle.
30What is the difference between absolute and fixed positioning? ▼
absolute positions an element relative to the nearest positioned ancestor; fixed positions it relative to the browser window.
31Explain opacity in CSS. ▼
The opacity property sets the transparency level of an element, between 0 (transparent) and 1 (opaque).
32What is the default value of the z-index property? ▼
The default is auto.
33How do you hide elements in CSS? ▼
Use display: none; or visibility: hidden;
34What properties are inherited by default? ▼
Properties like color, font-family, and font-size are inherited; layout properties like margin are usually not.
35What is @import in CSS? ▼
@import allows you to import style rules from other style sheets into a CSS file.
36What are vendor prefixes in CSS? ▼
Vendor prefixes (e.g., -webkit-, -moz-, -ms-, -o-) ensure compatibility with different browser engines for new features.
37Explain the use of calc() in CSS. ▼
calc() lets you perform calculations to determine CSS property values, e.g., width: calc(100% - 80px);
38What is a combinator selector? ▼
Combinators are used to combine two or more selectors, such as descendant (space), child (>), adjacent sibling (+), and general sibling (~).
39How do you create a responsive image? ▼
Set max-width: 100%; and height: auto; on the img element.
40What is the use of box-sizing property? ▼
box-sizing controls how the total width and height of an element is calculated. border-box includes padding and border in the element’s width and height.
41How do you clear floats? ▼
Use clear: both; on a following element or use a clearfix hack.
42What is shadow DOM in CSS? ▼
Shadow DOM allows encapsulation of CSS and markup so that styles are scoped to a specific element and not affected by external styles.
43What is remapping in CSS? ▼
Remapping generally refers to reassignment of values or typography scale within the root element using rem units.
44How do you animate elements in CSS? ▼
Use keyframes and the animation or transition properties.
45How do you make a triangle in CSS? ▼
Use the border property: set three borders to transparent and one to a color, with zero width or height on the element.
46What is the clamp() function in CSS? ▼
clamp() allows you to set a value between a specified minimum and maximum, e.g., font-size: clamp(1rem, 2vw, 2rem);
47How do you make text overflow ellipsis? ▼
Use white-space: nowrap; overflow: hidden; and text-overflow: ellipsis;
48What is the minmax() function in CSS Grid? ▼
minmax() defines the size range of a grid track, e.g., grid-template-columns: minmax(100px, 1fr);
49What is CSS isolation? ▼
The isolation property creates a new stacking context, preventing elements from overlapping incorrectly.
50What is the difference between :nth-child() and :nth-of-type()? ▼
:nth-child() selects the nth child regardless of type; :nth-of-type() only matches elements of the specified type.

Artificial Intelligence Interview Questions and Answers

1What is Artificial Intelligence (AI)?▼
Artificial Intelligence is the capability of a machine to imitate intelligent human behavior, including learning, reasoning, problem-solving, and perception.
2What are the different types of AI?▼
Types of AI include Narrow AI (Weak AI), General AI, and Superintelligent AI; based on functionalities: Reactive Machines, Limited Memory, Theory of Mind, and Self-aware AI.
3What is the Turing Test?▼
The Turing Test, developed by Alan Turing, measures a machine’s ability to exhibit intelligent behavior equivalent to, or indistinguishable from, that of a human.
4Define machine learning.▼
Machine Learning is a subfield of AI that enables machines to learn from data and improve performance without explicit programming.
5What are neural networks?▼
Neural networks are computing systems inspired by the human brain’s interconnected neuron structure, used to model complex patterns and prediction problems.
6Explain supervised vs. unsupervised learning.▼
Supervised learning uses labeled data to map input to output; unsupervised learning finds patterns or groupings in unlabeled data.
7What is deep learning?▼
Deep learning is a subset of machine learning utilizing multi-layered neural networks to model complex patterns in data.
8What are common applications of AI?▼
AI is used in natural language processing, computer vision, robotics, expert systems, speech recognition, and recommendation engines.
9What is reinforcement learning?▼
Reinforcement learning is a learning paradigm where agents learn to take actions to maximize cumulative rewards in an environment.
10What is natural language processing (NLP)?▼
NLP is an AI field focused on enabling machines to understand, interpret, and generate human language.
11What is computer vision?▼
Computer vision is an AI branch that enables computers to interpret and process visual information from the world, such as images and videos.
12What are expert systems?▼
Expert systems are computer programs that emulate the decision-making abilities of human experts in specific domains using rules and knowledge bases.
13What is a chatbot?▼
A chatbot is an AI-powered software program that conducts online chats via text or voice, simulating human conversation.
14Explain the term "overfitting" in machine learning.▼
Overfitting refers to a model that learns both data and noise, performing well on training data but poorly on unseen data.
15What is underfitting in machine learning?▼
Underfitting means the model is too simple to capture the underlying patterns in the data, resulting in poor performance on both training and test sets.
16What is transfer learning?▼
Transfer learning leverages knowledge from a model trained on one task to improve learning on a related task.
17Describe a confusion matrix.▼
A confusion matrix is a table used to evaluate classification models, showing true positives, true negatives, false positives, and false negatives.
18What is accuracy in machine learning?▼
Accuracy is the ratio of correctly predicted observations to the total observations, often used as a metric for classification models.
19What is precision and recall?▼
Precision is the ratio of true positives to all positive predictions; recall is the ratio of true positives to all actual positives.
20What is F1 score?▼
The F1 score is the harmonic mean of precision and recall, providing a balance between them for model evaluation.
21What is regularization?▼
Regularization is a technique in machine learning that discourages model complexity to prevent overfitting, using methods like L1 and L2 regularization.
22What are the differences between AI and ML?▼
AI is a broader concept for developing smart machines; ML is a subset of AI that allows systems to learn from data automatically.
23What is the difference between strong AI and weak AI?▼
Strong AI possesses genuine intelligence and consciousness; weak AI specializes in specific tasks without human-like understanding.
24Explain generative adversarial networks (GANs).▼
GANs are neural architectures consisting of a generator and a discriminator competing in a zero-sum game to produce realistic synthetic data.
25What is feature engineering?▼
Feature engineering involves creating new input variables or transforming existing ones to improve model performance.
26Define dimensionality reduction.▼
Dimensionality reduction reduces the number of input variables in a dataset, often using methods like PCA or t-SNE, for simpler, faster models.
27What is a decision tree?▼
A decision tree is a supervised learning algorithm used for classification and regression, representing decisions with branching nodes.
28What is ensemble learning?▼
Ensemble learning combines multiple models to improve accuracy and robustness over a single model, e.g., bagging, boosting, stacking.
29What is bias-variance tradeoff?▼
It refers to the balance between a model’s simplicity (bias) and its sensitivity to fluctuations in the training data (variance).
30What is the role of activation functions in neural networks?▼
Activation functions introduce non-linearity into neural networks, enabling them to learn and perform complex tasks. Examples: ReLU, sigmoid, tanh.
31What is dropout in deep learning?▼
Dropout is a regularization method where randomly selected neurons are ignored during training, reducing overfitting.
32What is backpropagation?▼
Backpropagation is a training algorithm for neural networks that computes gradients to update model parameters, enabling learning.
33Define convolutional neural networks (CNNs).▼
CNNs are deep learning models effective for image processing, utilizing convolutional layers to extract spatial data features.
34What are recurrent neural networks (RNNs)?▼
RNNs are neural networks suitable for sequential data, like time series or text, as they maintain internal memory over input sequences.
35What is reinforcement signal (reward)?▼
A reinforcement signal or reward is feedback provided to an agent in reinforcement learning to guide its actions toward desired outcomes.
36What is unsupervised clustering?▼
Clustering is grouping unlabeled data points based on similarity using algorithms like K-means or hierarchical clustering.
37What is K-means clustering?▼
K-means is an unsupervised algorithm that partitions data into K clusters by minimizing intra-cluster variance.
38What is a confusion matrix used for?▼
It's used for evaluating classification model performance by comparing predicted and actual values across all classes.
39What is a loss function?▼
A loss function measures the difference between predicted and actual outputs, guiding model optimization.
40Explain the difference between shallow and deep networks.▼
Shallow networks have one or two hidden layers; deep networks have many, enabling them to learn complex representations.
41What are support vector machines (SVM)?▼
SVMs are supervised learning models for classification and regression, finding the hyperplane that best divides classes.
42What is a perceptron?▼
A perceptron is the basic unit of a neural network that performs binary classification using a linear function and activation.
43What is data augmentation?▼
Data augmentation artificially increases the size of training data by creating modified versions, improving model generalization.
44What is an epoch in deep learning?▼
An epoch is one complete pass through the entire training dataset during model training.
45Define transfer learning.▼
Transfer learning is applying knowledge from one trained model to a new problem, often by reusing weights or layers.
46What is natural language understanding (NLU)?▼
NLU is a subfield of NLP focused on machine comprehension and interpretation of human language.
47What is natural language generation (NLG)?▼
NLG is the process of generating meaningful phrases and sentences in human language from machine representations.
48What are ethical concerns in AI?▼
Ethical concerns include bias, transparency, accountability, privacy, consent, and the potential societal impact of AI decisions.
49What are decision boundaries in classification?▼
Decision boundaries are lines or surfaces that separate different classes in feature space, created by classifiers.
50What is model interpretability?▼
Model interpretability is the degree to which a human can understand the cause of a decision made by an AI model.

Salesforce Interview Questions and Answers

1What is Salesforce?▼
Salesforce is a cloud-based customer relationship management (CRM) platform used to manage sales, customer service, marketing automation, analytics, and application development.
2What are the different types of Salesforce clouds?▼
Salesforce Clouds include Sales Cloud, Service Cloud, Marketing Cloud, Commerce Cloud, and Community Cloud.
3What is Apex?▼
Apex is a strongly typed, object-oriented programming language used on the Salesforce platform for executing flow and transaction control statements on Salesforce servers.
4What is a trigger in Salesforce?▼
A trigger is Apex code that executes before or after record operations like insert, update, delete, or undelete to perform custom actions.
5What are governor limits in Salesforce?▼
Governor limits are runtime limits enforced by Salesforce to ensure efficient resource usage and to prevent monopolization of shared resources.
6What is a Workflow in Salesforce?▼
Workflow is an automation tool that performs automated actions like field updates, email alerts, task creation, or outbound messages based on defined criteria.
7Explain the difference between a Role and Profile in Salesforce.▼
Profiles define user permissions and access to objects and fields; Roles control record-level access through the role hierarchy.
8What is a Salesforce Object?▼
